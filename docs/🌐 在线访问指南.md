# 🌐 在线访问指南

让别人也能访问这个工具的几种方案：

---

## 方案 1: GitHub Pages（推荐）⭐⭐⭐⭐⭐

### 优点
- ✅ 完全免费
- ✅ 自动部署（每次 push 自动更新）
- ✅ 稳定可靠
- ✅ 支持自定义域名

### 缺点
- ⚠️ 国内访问可能稍慢
- ⚠️ 只能部署静态页面

### 部署步骤

#### 1. 启用 GitHub Pages
1. 打开仓库：**https://github.com/lingdujiang/EnglishLearningTool**
2. 点击 **Settings**（设置）
3. 左侧菜单点击 **Pages**
4. **Source** 选择：
   - Deploy from a branch
5. **Branch** 选择：
   - `main` 分支
   - 文件夹选择 `/(root)` 或 `/code`（推荐 root）
6. 点击 **Save**

#### 2. 等待部署
- GitHub 会自动构建（约 1-2 分钟）
- 成功后会显示访问地址
- 格式：`https://用户名.github.io/仓库名/`

#### 3. 访问
部署成功后，访问：
```
https://lingdujiang.github.io/EnglishLearningTool/
```

## 方案 2: Vercel（推荐）⭐⭐⭐⭐⭐

### 优点
- ✅ 免费
- ✅ 自动部署
- ✅ 国内访问较快
- ✅ 支持自定义域名

### 部署步骤

1. **访问**: https://vercel.com
2. **登录**: 使用 GitHub 账号登录
3. **导入项目**: 
   - 点击 "Add New Project"
   - 选择 "Import Git Repository"
   - 选择 `EnglishLearningTool`
4. **配置**:
   - Framework Preset: `Other`
   - Root Directory: `code`
   - 其他默认
5. **部署**: 点击 "Deploy"

### 访问地址
```
https://english-learning-tool-xxx.vercel.app
```

---

## 方案 3: Netlify（推荐）⭐⭐⭐⭐⭐

### 优点
- ✅ 免费
- ✅ 拖拽部署
- ✅ 自动 HTTPS

### 部署步骤

1. **访问**: https://netlify.com
2. **注册登录**
3. **创建站点**:
   - 选择 "Add new site" → "Deploy manually"
   - 拖拽 `code` 文件夹到上传区域
4. **完成**

### 访问地址
```
https://xxx-xxx.netlify.app
```

---

## 方案 4: Gitee Pages（国内专用）⭐⭐⭐⭐

### 优点
- ✅ 国内访问快
- ✅ 免费

### 缺点
- ⚠️ 需要实名认证
- ⚠️ 需要审核

### 部署步骤

1. **同步到 Gitee**
```bash
git remote add gitee git@gitee.com:用户名/EnglishLearningTool.git
git push gitee main
```

2. **启用 Gitee Pages**
- 打开 https://gitee.com/用户名/EnglishLearningTool
- 设置 → Pages 服务
- 选择分支 → 提交

## 🔧 故障排查

### GitHub Pages 404
- 检查 Settings → Pages 是否启用
- 确认分支选择正确
- 等待 2-5 分钟部署完成

### 资源加载失败
- 检查路径是否为相对路径
- 确认 `code/` 目录结构正确

### 音频无法播放
- 检查音频文件格式
- 确认文件大小（GitHub 限制 100MB）

---

## 💡 最佳实践

1. **使用相对路径** - 确保在任何域名下都能访问
2. **压缩资源** - 图片、音频文件不要太大
3. **添加 PWA 支持** - 可以离线使用
4. **自定义域名** - 更专业、易记
