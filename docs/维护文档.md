## 📝 常见修改

### 修改词库

**文件**：`code/data/words.js`

**格式**：
```javascript
{
  id: 1,
  word: "cat",              // 单词
  meaning: "猫",            // 中文意思
  image: "images/cat.png",  // 图片路径
  audio: "audio/cat.wav",   // 音频路径
  sentence: "This is a cat." // 例句
}
```

**步骤**：
1. 编辑 `words.js`
2. 添加/修改单词
3. 准备对应的图片和音频
4. 刷新页面

---

### 修改动物等级规则

**文件**：`code/script.js`

**函数**：`getAnimalLevel(days)`

**默认规则**：
```javascript
if (days >= 10) return 4;   // 凤凰
if (days >= 6) return 3;    // 大鸟
if (days >= 3) return 2;    // 小鸟
if (days >= 1) return 1;    // 雏鸟
return 0;                    // 蛋
```

**修改后**：刷新页面生效。

---

### 切换布局样式 ⭐ NEW

**方法 1：通过设置面板**
1. 点击右上角 ⚙️ 按钮
2. 选择喜欢的布局（A/B/C）
3. 自动保存并应用

**方法 2：通过控制台**
```javascript
// 设置为紧凑布局
setLayout('layout-a')

// 设置为左右分栏
setLayout('layout-b')

// 设置为卡片悬浮
setLayout('layout-c')
```

**方法 3：直接修改 localStorage**
```javascript
// 在控制台输入
localStorage.setItem('layout', 'layout-a')
location.reload()
```

---

### 重置学习进度 ⭐ NEW

**方法 1：通过重置按钮**
1. 点击底部 🗑️ 重置进度 按钮
2. 确认弹窗中点击"确定"
3. 进度清除，页面刷新

**方法 2：通过控制台**
```javascript

resetProgress()
```

**方法 3：清除 localStorage**
```javascript
localStorage.clear()
location.reload()
```

### 修改每日单词数量

**文件**：`code/script.js`

**位置**：多处使用数字 `5` 的地方

**需要修改的位置**：
1. `checkTodayStatus()` - 判断是否完成

**当前**：小鸡成长（蛋→雏鸟→小鸟→大鸟→凤凰）

**可替换为**：
- 🐛 毛毛虫→蝴蝶
- 🌱 种子→大树
- 🐸 蝌蚪→青蛙
- 🚀 火箭发射阶段

**步骤**：
1. 准备 5 张新图片
2. 命名为 `egg.png`, `chick.png`, `bird.png`, `bigbird.png`, `phoenix.png`
3. 放入 `images/` 文件夹
4. 修改 `script.js` 中的动物名称显示

## 🐛 故障排查

### 问题 1：图片不显示

**症状**：动物等级图或单词图片显示为空白

**排查步骤**：
1. 按 F12 打开控制台
2. 查看是否有 404 错误
3. 检查图片文件是否存在
4. 强制刷新页面（Ctrl + F5）

**解决**：
```bash
# 重新下载图片
powershell -ExecutionPolicy Bypass -File "code/quick-download-images.ps1"
```

---

### 问题 2：声音不播放

**症状**：点击发音按钮没声音

**排查步骤**：
1. 检查电脑音量
2. 检查浏览器是否静音
3. 按 F12 查看控制台错误
4. 检查 `audio/` 文件夹是否有 WAV 文件

**解决**：
```bash
# 重新生成音频
powershell -ExecutionPolicy Bypass -File "code/generate-audio.ps1"
```

---

### 问题 3：进度不保存

**症状**：刷新页面后学习进度丢失

**排查步骤**：
1. 按 F12 打开控制台
2. 输入 `localStorage` 查看存储内容
3. 检查浏览器是否禁用本地存储

**解决**：
- 清除浏览器缓存
- 确保使用现代浏览器（Chrome/Edge）

---

### 问题 4：第二天不更新单词

**症状**：第二天打开还是昨天的单词

**排查步骤**：
1. 检查系统日期是否正确
2. 按 F12 查看控制台日志
3. 检查 `lastLearnDate` 存储

**手动重置**：
```javascript
// 在控制台输入
localStorage.clear()
location.reload()
```

---

### 问题 5：布局切换后显示异常 ⭐ NEW

**症状**：切换布局后页面显示错乱

**排查步骤**：
1. 检查浏览器控制台是否有 CSS 错误
2. 清除浏览器缓存（Ctrl + Shift + Delete）
3. 强制刷新页面（Ctrl + F5）

**解决**：
```javascript
// 重置为默认布局（紧凑布局）
localStorage.setItem('layout', 'layout-a')
location.reload()
```

---

### 问题 6：设置面板无法打开 ⭐ NEW

**症状**：点击设置按钮无反应

**排查步骤**：
1. 按 F12 查看控制台错误
2. 检查 `btn-settings` 元素是否存在
3. 检查 JavaScript 是否加载完成

**解决**：
- 强制刷新页面
- 清除浏览器缓存
- 检查浏览器控制台错误信息

### 常见错误代码

|------|------|------|

## 🎯 最佳实践

### 性能优化
- 图片大小控制在 50-200KB
- 使用 PNG 格式（透明背景）
- 音频使用 WAV 或 MP3

- 按钮尺寸适合儿童点击

### 安全注意

- 备份重要数据
- 定期清理缓存
