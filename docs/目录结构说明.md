## 📋 文件分类说明

### ✅ 正式程序（code/）
这些是运行程序必需的文件，**不要删除**：
- `index.html` - 主页面
- `style.css` - 样式
- `script.js` - 逻辑
- `data/words.js` - 词库
- `images/` - 所有图片
- `audio/` - 所有音频

### 🛠️ 工具脚本（tools/）
用于维护和管理，**偶尔使用**：
- `quick-download-images.ps1` - 下载图片
- `generate-audio.ps1` - 生成音频
- `check-images.ps1` - 检查文件

### 🔍 调试工具（tools/debug/）
用于问题排查，**一般不用**：
- 所有 `test-*.html` 文件
- 用于诊断图片、声音、TTS 等问题

### 📖 文档（docs/）
说明和维护指南：
- 需求文档
- 维护文档
- 本文件

## 🚀 快速开始

### 使用程序
```
1. 打开 code/index.html
2. 开始学习！
```

### 下载图片
```bash
powershell -ExecutionPolicy Bypass -File "tools/quick-download-images.ps1"
```

### 生成音频
```bash
powershell -ExecutionPolicy Bypass -File "tools/generate-audio.ps1"
```

### 检查文件
```bash
powershell -ExecutionPolicy Bypass -File "tools/check-images.ps1"
```

## 📦 备份建议

### 每周备份
```bash
# 备份整个项目
Compress-Archive -Path "D:\clawProject\EnglishLearningProject" -DestinationPath "backup\backup-$(Get-Date -Format 'yyyyMMdd').zip"
```

### 备份位置
所有备份文件放在 `backup/` 文件夹中。
