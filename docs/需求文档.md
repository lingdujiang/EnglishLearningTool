# 📄 产品需求文档 - 每日 5 词 MVP

## 1. 产品概述

**产品名称**：（待定，比如「单词小动物园」）  
**目标用户**：小学一年级学生  
**核心价值**：每天学 5 个单词，用小动物成长激励坚持  
**验证目标**：孩子是否愿意每天主动打开学习

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 每日 5 词 | 每天展示 5 个单词，学完解锁下一个 | P0 |
| 听发音 | 点击播放单词标准发音 | P0 |
| 跟读 | 孩子跟读，录音播放对比（可选） | P1 |
| 上一词 | 点击返回上一个单词复习（第 1 词时禁用） | P1 |
| 下一词 | 学完点击进入下一个单词 | P0 |
| 小动物成长 | 完成 5 词后小动物升级 | P0 |

### 2.2 功能流程

```
打开网页
  ↓
显示今日第 1 词（图片 + 单词 + 发音按钮 + 上一词/下一词按钮）
  ↓
点击发音 → 播放音频
  ↓
孩子跟读（可选录音）
  ↓
点击「下一个」→ 第 2 词
  ↓
点击「上一个」→ 返回第 1 词复习（可选）
  ↓
... 重复 5 次
  ↓
完成！小动物长大 🐣→🐤→🐔
  ↓
显示鼓励语 + 明天再来
```

## 4. 界面设计

### 4.1 页面结构

```
单页面应用，3 个状态：

1. 学习页（默认）
   ┌─────────────────┐
   │   🐣 等级显示    │
   ├─────────────────┤
   │                 │
   │    [图片]       │
   │                 │
   │     cat         │
   │     猫          │
   │                 │
   │  🔊 [发音]      │
   │  🎤 [跟读]      │
   │                 │
   │  [← 上一个]     │
   │  [下一个 →]     │
   │                 │
   │  进度：●○○○○    │
   └─────────────────┘
   
   注：第 1 词时「上一个」按钮禁用（灰色）

2. 完成页（5 词学完）
   ┌─────────────────┐
   │   🎉 太棒了！    │
   │                 │
   │   小动物长大了  │
   │     🐣→🐤       │
   │                 │
   │  [明天继续]     │
   └─────────────────┘

3. 已完成页（今天学完了）
   ┌─────────────────┐
   │   ✅ 今天已完成  │
   │                 │
   │   明天再来哦    │
   │                 │
   │  [🔄 再来一次]  │
   │  [复习旧词]     │
   └─────────────────┘
   
   注：「再来一次」重置今日进度，可重新学习
```

### 4.2 视觉风格

- **配色**：明亮活泼（黄/蓝/绿）
- **字体**：大号、圆润（适合儿童）
- **按钮**：大尺寸、易点击
- **动画**：简单过渡效果

| 组件 | 选择 | 理由 |

### 5.3 关键逻辑

```javascript
// 1. 判断今天是否已完成
function checkTodayStatus() {
  const lastDate = localStorage.getItem('lastLearnDate');
  const today = new Date().toDateString();
  
  if (lastDate === today) {
    const todayIndex = localStorage.getItem('todayIndex');
    if (todayIndex >= 5) {
      showCompletedPage();
      return;
    }
  } else {
    // 新的一天，重置进度
    localStorage.setItem('lastLearnDate', today);
    localStorage.setItem('todayIndex', 0);
  }
  
  showLearningPage();
}

// 2. 播放发音
function playAudio(word) {
  const audio = new Audio(`/audio/${word}.mp3`);
  audio.play();
}

// 3. 更新动物等级
function updateAnimalLevel() {
  const days = parseInt(localStorage.getItem('totalDays')) || 0;
  let level = 0;
  if (days >= 10) level = 4;
  else if (days >= 6) level = 3;
  else if (days >= 3) level = 2;
  else if (days >= 1) level = 1;
  
  document.getElementById('animal').src = `/images/level${level}.png`;
}
```
