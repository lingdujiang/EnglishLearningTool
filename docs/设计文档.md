## 产品概述

### 产品名称
单词小动物园（Word Zoo）

### 产品定位
一款面向小学一年级学生的英语单词学习工具，通过动物成长系统激励学生每日学习。

### 核心价值
- 🎯 **每日 5 词**：适量学习，减轻负担
- 🐣 **动物成长**：游戏化激励，培养习惯
- 🔊 **听读结合**：发音 + 跟读，全面提升
- 📊 **进度追踪**：可视化反馈，成就感满满

## 功能设计

### 核心功能

#### 1. 每日学习
- 每天学习 5 个单词
- 自动记录学习进度
- 完成后解锁第二天内容

#### 2. 动物成长系统
- 5 个等级：蛋 → 雏鸟 → 小鸟 → 大鸟 → 凤凰
- 根据连续学习天数升级
- 升级时显示动画效果

#### 3. 发音功能
- 🔊 听发音：播放单词音频
- 🎤 跟读：录音并回放对比

#### 4. 进度管理
- 今日进度：5 个进度点
- 连续天数统计
- 总学习单词数统计

### 新增功能（v1.3）

#### 5. 布局切换 ⭐ NEW
- **方案 A - 紧凑布局**：垂直空间压缩，一屏显示
- **方案 B - 左右分栏**：图片在左，文字在右
- **方案 C - 卡片悬浮**：背景透明，卡片悬浮中央
- 设置面板自由选择
- 布局自动保存

#### 6. 重置进度 ⭐ NEW
- 底部重置按钮
- 确认弹窗保护
- 清除所有学习数据

### 三种布局方案

#### 布局 A：紧凑布局
```
特点：
- 所有元素垂直排列
- 字体和图片缩小
- 减少间距和 padding
- 适合小屏幕设备

尺寸调整：
- 头部：160px → 70px
- 动物图片：120px → 50px
- 单词图片：200px → 120px
- 单词字体：48px → 32px
- 释义字体：24px → 18px
```

#### 布局 B：左右分栏
```
特点：
- 左右两栏布局
- 图片在左，文字在右
- 横向利用空间
- 适合宽屏设备

布局结构：
┌─────────┬──────────────┐
│  图片   │   单词文本    │
│ 140px   │   释义        │
│         │   按钮组      │
├─────────┴──────────────┤
│      进度条             │
└─────────────────────────┘
```

#### 布局 C：卡片悬浮
```
特点：
- 渐变背景全屏
- 白色卡片悬浮
- 动物等级半透明
- 视觉效果最佳

视觉层次：
- 背景层：渐变色彩
- 内容层：白色卡片（阴影）
- 悬浮层：动物等级、进度条
```

```javascript
{
  id: 1,                    // 唯一 ID
  word: "cat",              // 英文单词
  meaning: "猫",            // 中文释义
  image: "images/cat.png",  // 图片路径
  audio: "audio/cat.mp3",   // 音频路径
  sentence: "This is a cat." // 例句
}
```

### 动物等级配置

```javascript
const animalImages = [
  'images/egg_0.png',   // 等级 0：蛋
  'images/egg_1.png',   // 等级 1：雏鸟
  'images/egg_2.png',   // 等级 2：小鸟
  'images/egg_3.png',   // 等级 3：大鸟
  'images/egg_4.png'    // 等级 4：凤凰
];

const animalNames = ['蛋', '雏鸟', '小鸟', '大鸟', '凤凰'];

// 等级规则（连续学习天数）
function getAnimalLevel(days) {
  if (days >= 10) return 4;   // 凤凰
  if (days >= 6) return 3;    // 大鸟
  if (days >= 3) return 2;    // 小鸟
  if (days >= 1) return 1;    // 雏鸟
  return 0;                    // 蛋
}
```

## 交互设计

### 用户流程

```
打开应用
    ↓
检查今日状态
    ↓
显示单词卡片
    ↓
┌───────────────┐
│  听发音/跟读  │ ← 可重复
└───────────────┘
    ↓
点击"下一个"
    ↓
完成 5 个单词？
    ↓
是 → 显示完成页面 → 明天继续
否 → 继续学习
```

### 按钮交互

| 按钮 | 功能 | 交互 |
|------|------|------|
| 🔊 听发音 | 播放单词音频 | 点击即播放 |
| 🎤 跟读 | 录音并回放 | 点击开始/停止 |
| ← 上一个 | 返回上一个单词 | 第一个单词时禁用 |
| 下一个 → | 进入下一个单词 | 完成后显示升级 |
| ⚙️ 设置 | 打开布局设置 | 弹出设置面板 |
| 🗑️ 重置 | 重置学习进度 | 二次确认 |

### 动画效果

```css
/* 页面切换 */
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

/* 升级动画 */
@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.1); }
}

/* 录音状态 */
@keyframes recording {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.6; }
}

/* 设置面板 */
@keyframes slideUp {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}
```
