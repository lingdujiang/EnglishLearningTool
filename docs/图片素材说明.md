# 🎨 图片素材生成指南

## ✅ 已生成的文件

项目中已包含以下图片生成工具：

### 1. create-images.html（推荐）
**完全离线生成 PNG 图片**

**使用方法**:
1. 双击打开 `code/create-images.html`
2. 点击「📥 一键下载所有图片」按钮
3. 浏览器会自动下载 55 张 PNG 图片
4. 在 Downloads 文件夹中找到图片
5. 将所有图片复制到 `code/images/` 文件夹

**生成的图片**:
- 动物等级图：5 张（egg.png, chick.png, bird.png, bigbird.png, phoenix.png）
- 单词图片：50 张（cat.png, dog.png, ...）

---

### 2. download-images.ps1
**PowerShell 脚本，从网络下载**

**使用方法**:
```bash
powershell -ExecutionPolicy Bypass -File "code/download-images.ps1"
```

**注意**: 需要联网，使用占位图服务生成带颜色的图片。

---

### 3. generate-audio.ps1
**PowerShell 脚本，生成音频文件**

**使用方法**:
```bash
powershell -ExecutionPolicy Bypass -File "code/generate-audio.ps1"
```

**生成**: 50 个 WAV 音频文件到 `code/audio/` 文件夹

## 🚀 快速完成步骤

### 第 1 步：生成音频文件

```bash
powershell -ExecutionPolicy Bypass -File "code/generate-audio.ps1"
```

等待 1-2 分钟，生成 50 个 WAV 文件。

### 第 2 步：生成图片素材

1. 打开 `code/create-images.html`
2. 点击「📥 一键下载所有图片」
3. 等待浏览器下载 55 张图片
4. 将所有图片移动到 `code/images/` 文件夹

### 第 3 步：测试

1. 打开 `code/index.html`
2. 点击「🔊 听发音」按钮 - 应该能听到声音
3. 检查图片是否正常显示
4. 完成！🎉

## ⚠️ 注意事项

1. **图片格式**: 支持 PNG、JPG、SVG
2. **图片大小**: 建议 200x200 到 400x400 像素
3. **文件命名**: 必须与单词一致（小写）
4. **如果图片不显示**: 按 F12 查看控制台错误信息

---

## 🎨 自定义图片

如果想用自己的图片：

1. 准备 55 张图片
2. 命名为对应的文件名（如 cat.png）
3. 放入 `code/images/` 文件夹
4. 刷新页面即可

---

## 🔧 故障排查

### 图片不显示
1. 检查文件是否在正确的文件夹
2. 检查文件名是否正确（小写，无空格）
3. 按 F12 查看控制台是否有 404 错误

### 音频不播放
1. 检查 `audio/` 文件夹是否存在
2. 检查是否有对应的 WAV/MP3 文件
3. 按 F12 查看控制台错误信息
